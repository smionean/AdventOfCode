# Défi 2025 en Ada 
(https://adventofcode.com/2025)

Pour créer le projet de la journée (exemple : jour 01)
```
template/generate.sh 01
```

Pour compiler
```
gprbuild day01.gpr
```

Pour exécuter
```
obj/day01
```
