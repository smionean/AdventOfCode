Pour créer le projet de la journée (exemple : jour 01) / Create 
```
template/generate.sh 01
```

## Pour Jour 1 et 3 / Day 1 and 3
Compiler / Build 
```
gprbuild day01.gpr
```

Exécuter / Run
```
obj/day01
```

## Pour Jour 4 et après / Day 4 and later
[![Static Badge](https://img.shields.io/badge/Build_with-Alire-blue)](https://alire.ada.dev/)

Compiler et exécuter / Build and run :
```
$ alr run
```
