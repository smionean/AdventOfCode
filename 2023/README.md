# Défi 2023 en Ada 
(https://adventofcode.com/2023)

Pour créer le projet de la journée (exemple : jour 01)
```
template/generate.sh 02
```

Pour compiler
```
gprbuild day01.gpr
```

Pour exécuter
```
obj/day01
```
